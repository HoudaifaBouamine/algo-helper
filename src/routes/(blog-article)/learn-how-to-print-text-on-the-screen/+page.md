#### Learn Text Output (Write, WriteLn)

---

### Problem #1

Write a program that print "Hello Computer Sience !!!"

### Final Solution

<CodeBlock lang="pascal">

```pascal
program HelloCS;

begin
  Write('Hello Computer Sience !!!');
end.
```

</CodeBlock>

### Program Output

<CodeBlock lang="shell">

```shell
Hello Computer Sience !!!
```

</CodeBlock>
